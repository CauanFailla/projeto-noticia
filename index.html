Chat GPT

ChatGPT é um chatbot que usa inteligência artificial (IA) para conversar com os usuários, e até mesmo criar códigos de diversas linguagem de programação.

Desenvolvido pela startup OpenAI, também responsável pelo Dall-E, o robô funciona como uma espécie de Alexa, só que em texto: o usuário faz uma pergunta ou emite um comando, e o ChatGPT responde dentro de alguns segundos.

Objetivos

Segundo seus criadores, o objetivo, é o de reproduzir cada vez mais os modelos de interação humana. Além de interagir com as pessoas e trazer respostas para perguntas, o sistema é capaz de criar codigos de programação, músicas, roteiros, artigos e até poemas - e a internet se diverte. Têm viralizado casos de usuários que testaram o potencial de criação do sistema, como a possibilidade de criar uma peça para piano no estilo de Mozart.

Como funciona o Chat GPT?

O ChatGPT é um chatbot conectado à internet. O robô pega informações que encontra, para poder dar suas respostas. O modelo funciona por meio de aprendizagem de máquina, ou Machine Learning, como também é conhecido esse processo.

A aprendizagem de máquina é considerada como o futuro da computação. Já que pode criar Inteligências Artificiais que cheguem próximo ao ser humano. Assim, cada vez mais essa tecnologia está se aprimorando.
